
# Task 1A

This task aims to implement a drawing node for turtlesim to draw the symbol '8'(eight).

### **Build the workspace**
```bash
cd ~/Hologlyph-Bots/hb_task1a_ws/
colcon build
source install/setup.bash
```

### **Launch Turtlesim**
```bash
ros2 run turtlesim turtlesim_node
```
